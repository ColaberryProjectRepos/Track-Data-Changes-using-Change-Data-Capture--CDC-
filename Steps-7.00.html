How to implement a Data Manipulation language(DML) to CDC ?

Here are some examples of  using DML in order to manipulate the data  and how CDC track them. 

These are operation meaning and descriptions. 

Column _$operation contains value which corresponds to DML Operations. Following is quick list of value and its corresponding meaning.

  • Delete Statement = 1

  • Insert Statement = 2

  • Value before Update Statement = 3

  • Value after Update Statement = 4

On the examples below,  cdc.dbo_Glob_store_CT will capture all the activities of Inserted, Updated, and Deleted of table name dbo.Glob_store. When there is any change  made on a dbo.Glob_store table, it will track all the changes.  The below  screenshots show example of change data capture when the records are updated.updated.

The below query will change a customer Name Aaron BG into Aaron Bergman, that is just to show you how that change is captured.
